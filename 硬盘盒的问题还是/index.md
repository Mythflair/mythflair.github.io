# 硬盘盒的问题还是?


- YOTTAMASTER 双盘位硬盘盒，西部数据 4.0T 红盘和希捷 4.0T 红盘，主要存些杂项文件和游戏，没有电影。
- 西数红盘已经“挂掉”两次了，说是挂掉，就是莫名其妙的变为未分配状态，用简单的硬盘检测工具，发现状态正常，但都是在装了 windows 11 后出现的。
- 按说操作系统导致这种情况的可能性极小。
- 今天莫名其妙的，希捷也提示“未分配”状态，折腾许久，换了个多年前的裸族硬盘坞，希捷显示出来了，状态正常，西数仍然处于未分配状态。
- 另外就是，希捷的工作时间为 1万 小时左右，而西数的工作时间为 1.6万 小时左右，同时启用的两块硬盘，有点儿匪夷所思。
- 这种问题，只能排除法，硬盘盒换掉了，再观察看看。
- 好在不是台式机内的重要数据盘，不然真不知道该怎么办。
- 继续观察。。。。。。



- 20210814 问题找到了，分区丢失，用傲梅硬盘助手->所有工具->恢复分区，就恢复正常了。只是可惜了第一次分区丢失，当时直接清空了硬盘，丢了哪些数据都不知道。
- 不过话说回来，丢了的数据，都不知道丢了哪些数据，也就等于这些数据其实也没那么重要吧。
- 另外，硬盘盒的原因还是硬盘的原因还不能确定，只能用这个旧的硬盘盒再测试一段，才能得出结论。
- 继续观察。。。。。。
