# 病了吗？


- 2021年9月28日早晨，发现大便最后，有血！~

- 吓坏了，用手机拍了照，一直处于紧张、焦虑、失眠当中。

- 2021年9月27日晚，和税务局的朋友一起吃饭，喝了大约流量白酒，五粮液56度、山庄16年38度、健康100 36度，各二两吧，后有同学邀约，又喝了有10瓶啤酒，28日凌晨1点多才回家。

- 虽然这么多年一直处于半酗酒状态，应酬多喝酒、没有应酬在家也要喝不少啤酒，近五年已经成了常态。之前主要是因为工作压力大，晚上无论多晚回家，也要喝点儿啤酒解压助眠。之后，就成了习惯，非常不好的习惯，不喝酒会很难受。
- 但，酒后便血，还是第一次发现，虽然之前觉得大便的最后，会有些稀，且颜色不太正常，但也不好描述。
- 从9月28日开始，一直处于紧张焦虑中，询问了医院的朋友，内痔到肠癌的判断，区间好大；百度更是不能查，随便搜索那都是绝症。果然，晚上失眠了。
- 想想，自己真的不是怕死，就是放不下妻儿老小。家庭经济不能算大富大贵，也能算是小康家庭，如果自己不在了，经济上不是特别担心，只是家里没有了成年男性，妻子一人辛苦不说，很多家务和琐事，也根本不是女性能做的。
- 想想两个孩子都还未成年、母亲年迈、妻子一人实难抗下这么大的摊子，就很心酸。
- 之前特别看重的所谓的各种应酬和社会交往，到如今却也轻如鸿毛、不值一提；那些所谓的人生追求和患得患失，在健康和生命面前，根本不值得去回忆。
- 莫名的，想不起之前为什么如此好酒，心态如此差劲。可能是因为没有追求和明确的目标吧，得过且过，迷茫无方向，导致总是用酒麻醉自己。另外，多少有些社恐，酒可以充分放松自己，打开话匣子，便于交流。但，以后这些肯定都不重要了。
- 29日早上，大便基本正常，只有一小条粘液带点儿红色，网上查了查说是也可以自行指检，就试着自行检查了一下，发现并没有摸到可疑的感觉，但依旧忧心忡忡。
- 经过28日、29日，两天的紧张、焦虑、失眠，加之没怎么喝水吃饭，体重成功下降四斤，真是减肥奇方，30日一早，起床体重仅为165.4斤。
- 29日晚22点早早睡下，四小时后又醒来，心中难过，30日凌晨2点，又开始胡乱搜索相关信息，直到改变了搜索方向，因之前搜索的都是便血，提供的信息大都是无故便血，结果都是往吃席方向的；今早搜索了酒后便血，把条件输入进去后，结论逐渐往让人想开些的方向走了，心情也好了不少，紧张得到一定程度的缓解。
- 再观察几天，酒以后是不喝了，如果给我机会的话，希望能够看到孩子成年、母亲送终。
- 希望这是我人生中一记深刻的教训，不再透支身体，即使不为自己，也为了自己所爱和爱自己的人。
- 财富多少、地位有无，都无所谓，人在家在，健康平安，就是最大的幸福。
