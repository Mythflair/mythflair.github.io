# 鱼和花


- 奄奄一息的鱼，放生了三条在河里，今年雨水大，都不知道被冲到哪里去了，生死未卜；还有四条鱼，养在了窗台的生态鱼缸里，一反之前养在不见阳光的客厅中的常态，状态出奇的好，活泼、鲜红、几个月不喂食，靠吃植物活着；
- 房间里的植物很多，看着不错、心情也很好，只是植物越多，需要大力的就越多；
- 地下室的摩托车，放了一年多了，当时兴奋的找朋友借来，只是一时兴起，真喜欢吗？应该不是，现在放在地下室，没有了电，无法启动，还挡路，增加了麻烦。
- 这世界，少即是多，生活也需要化繁为简。想要的越多，麻烦就会更多，这个循环并非无解，需要你超然到另一个维度，但并不容易，也不是所有人都能达到、都愿意达到。
